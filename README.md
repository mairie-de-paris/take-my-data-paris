take-my-data-paris
==================
The project "Take My Data Paris" consists of an Android application and the back office tool to manage the application.
The project is an experiment by the city of Paris. It allows the user to visualize datasets on a global map or on thematic maps.
The user can participate in improving and enriching the data (crowdsourcing) by collecting new data from their smartphone.
Contributors improve the datasets and make it fun to use by creating a community of users motivated by the same goals.

Le projet Take My Data Paris comprend une application Android et un back office de gestion de l'application Android.
Initié sous forme d'expérimentation par la Ville de Paris, cette application permet à l'utilisateur de visualiser des jeux de données sur une carte globale ou des cartes thématiques.
L'utilisateur peut prendre part à l'amélioration et l'enrichissement des données (crowdsourcing) en collectant de nouvelles données depuis son smartphone.
Un système de challenge et de classement des contributeurs (sur la base du volontariat) permet de dynamiser cette collecte et de la rendre ludique à l'échelle d'une communauté d'utilisateurs animés par les mêmes objectifs.
