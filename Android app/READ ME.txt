Quelques informations importantes pour la mise en service / mise a jour de l'application TakeMyData :


1) definition de l'api-key google map : afin de faire fonctionner la carte correctement,
il vous appartient de remplacer l'api-key actuelle avec la votre.
Vous pourrez la trouver dans le fichier res/layout/map.xml  La ligne a modifier est la suivante :

        android:apiKey="0wQkAs5bcXTczkjiFcvNduYry3w5zydzl89gR1Q"

	plus d'infos sur l'api google map : https://developers.google.com/maps/signup

2) mise a jour des textes : l'ensemble des elements textuels visible dans l'application sont definis dans un meme fichier
situe dans res/values/strings.xml . Aucun texte n'a ete place dans le code source de maniere a faciliter une possible traduction 
de l'application dans le future.

3) Requetes vers le serveur : l'application est pour le moment connectee a notre serveur. Il vous appartient donc de modifier
certains elements pour "connecter" l'appli au votre.
   - Modifier le fichier src/com/utils/MyAsyncTask et editer la ligne :
	private static final String SERVER_URL = "http://88.191.143.205";
	en remplacant "http://88.191.143.205" par l'url de votre serveur ou se situe la partie web de l'application (dossier tmd)

4) The application uses the Sherlock action bar open source project, you'll need to include it in the project, as a library to compile correctly
